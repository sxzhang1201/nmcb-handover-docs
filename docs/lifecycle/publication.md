# Data publication

This stage covers how NMCB data and findings are made public: publications, reports, and data sharing for external research.

## What belongs in publication

| Activity | Purpose |
|----------|---------|
| Scientific publications | Papers, preprints |
| Reports | Study reports, progress updates |
| Data sharing | Depositing data in repositories |
| Documentation | Published protocols, data dictionaries |

## Relationship to the pipeline

Publication typically follows:

- [Analysis](analysis.md) — Analytical results feed into manuscripts and reports
- [Issuing](issuing.md) — Data requests may lead to co-authored work or shared datasets

## Hand-over considerations

- Document where publication-related materials live (manuscripts, supplementary files, data dictionaries).
- Note approval routes: who reviews before submission or release.
- For data sharing, document the repository, version, and any access conditions.

## Governance

Publication and data sharing are governed by:

- [Data requests](../governance/data-requests.md) — Process for external access
- [SOPs](../governance/sops.md) — Approved procedures for release
