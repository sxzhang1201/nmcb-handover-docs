# myDRE

*Role: secure environment for sharing datasets with researchers.*

myDRE is used for data issuing: it provides a secure environment where data requestors can analyze data without receiving raw data exports.

## Purpose

- Secure analysis environment for approved data requestors
- Enables analysis without direct data transfer
- Supports controlled access and audit trails

## Handover priorities

Document:

- how access is provisioned
- what data or views are available in the environment
- connection and login procedures
- escalation for access or technical issues
