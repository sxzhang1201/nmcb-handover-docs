# Research Drive

*Role: shared file storage and study folder layout.*

The Research Drive is the primary file storage for NMCB data, scripts, and documentation.

## Purpose

- Raw and processed data storage
- Scripts and code
- Documentation and logs
- Outgoing deliverables

## Folder design principle

A good storage structure should separate:

- raw data
- processed data
- scripts
- logs
- documentation
- outgoing deliverables

## Handover priorities

Document:

- folder structure and naming conventions
- access and permissions
- backup and retention
- relationship to [Snowflake](snowflake.md) and other systems
