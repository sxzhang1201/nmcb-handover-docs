# Multicentre data

*Role: data transfer and coordination between study sites.*

The board shows work around multicentre collaboration and data transfer between sites.

## Board themes

- manual of procedures for NMCB
- sample processing data in Nijmegen
- lab data and alert in Nijmegen
- data transfer towards A-UMC (Amsterdam UMC)

## Handover priorities

For data moving between Nijmegen and Amsterdam UMC or other sites, document:

- transfer trigger
- transfer frequency
- allowed transport mechanism
- file naming and structure
- expected confirmation back from receiver
- escalation if files are missing or inconsistent

## Operational risk

Multicentre workflows are especially vulnerable to confusion when naming conventions, sample identifiers, or template versions are not tightly controlled.
