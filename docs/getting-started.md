# Scope of this hand-over

## Role summary

This role supports the NMCB study as a bridge between:

- study coordination
- data collection operations
- data management
- lightweight data engineering

The work is not only about keeping data tidy. It also includes designing workflows that reduce manual work, improve traceability, and make future data sharing easier. 

## In scope

The current hand-over covers work reflected in the task board, including:

- recurring operational tasks
- Castor logic, exports, and reminders
- device-related data handling and QC
- scripts for transformation and eligibility
- data structure design and Snowflake-related work
- biobank data flow and request procedures
- SOP and documentation development

## Out of scope

This hand-over does not attempt to fully document:

- all scientific decision-making behind variable definitions
- all institutional approval routes outside the project team
- deep infrastructure administration that sits fully with central IT

Those dependencies should still be captured in the relevant SOPs, project folders, and contact lists.

## Handover principle

The next person should be able to do the work without relying on memory-based explanations. For each process, the target is to document:

- trigger
- owner
- systems used
- inputs
- step-by-step actions
- output
- checks
- common failure points
